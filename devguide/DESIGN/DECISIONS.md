# Design Decisions

This file collects important design decisions (like lightweight ADRs).

Use the following structure for each decision:

---

## [YYYY-MM-DD] Title of the decision

- **Context:**  
  Brief description of the context and problem.

- **Options considered:**  
  - Option A  
  - Option B  

- **Decision:**  
  What was chosen and why.

- **Consequences:**  
  Positive and negative implications, and follow-up actions.

---

Add new sections chronologically as decisions are made.
